# 🧬 AI Data Analyst Pro (Purple Edition)

An interactive **AI-powered Data Analytics Dashboard** built with Streamlit. This agent allows users to query a dataset (CSV) using natural language or a predefined list of 155 analytical questions. It features a sleek purple gradient UI, automated charting, and deep data filtering.



## 🚀 Features

* **Natural Language Querying:** Ask questions about employees, salaries, ranks, and states.
* **155+ Predefined Queries:** A comprehensive list of analytical questions ranging from basic lookups to complex aggregations.
* **Interactive Visualizations:** Automatically generates Salary Distribution and State-wise employee count charts.
* **Purple Gradient UI:** Custom CSS styling for a modern, high-tech aesthetic.
* **Export Data:** Options to download filtered results or the entire dataset as CSV.
* **Smart Logic Engine:** Handles specific name searches, rank-based filtering, and state-specific analysis.

## 🛠️ Tech Stack

* **Frontend/App Framework:** [Streamlit](https://streamlit.io/)
* **Data Manipulation:** [Pandas](https://pandas.pydata.org/)
* **Visualization:** [Matplotlib](https://matplotlib.org/) & [Seaborn](https://seaborn.pydata.org/)
* **Language:** Python 3.x

## 📋 How to Use

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/AI-Data-Analyst-Pro.git](https://github.com/YOUR_USERNAME/AI-Data-Analyst-Pro.git)
    ```
2.  **Install Dependencies:**
    ```bash
    pip install streamlit pandas matplotlib seaborn
    ```
3.  **Setup Data:**
    Place your `Ai agent 1.csv` on your Desktop or update the `FILE_PATH` in the code.
4.  **Run the App:**
    ```bash
    streamlit run app.py
    ```
